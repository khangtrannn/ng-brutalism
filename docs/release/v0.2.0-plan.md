# v0.2.0 Release Plan — Composition Primitives

## Goal

> "With v0.2.0 of @ng-brutalism/ui you can build these 18 designs effortlessly."

All 18 reference designs in `apps/docs/public/design/` must be reproducible using v0.2.0 primitives at ~90% fidelity. Not pixel-perfect — every structural primitive exists in the library; decorative details like barcodes and mini charts ship as docs-only CSS/SVG recipes.

Release strategy: **batch release**. Everything ships together as one announcement.

## Primitives

### Already implemented — docs pages done

| Component | File | Docs |
|---|---|---|
| `NbStat` | `libs/ui/src/lib/stat/nb-stat.ts` | Done |
| `NbRating` | `libs/ui/src/lib/rating/nb-rating.ts` | Done |
| `NbMediaFrame` | `libs/ui/src/lib/media-frame/nb-media-frame.ts` | Done |

### New — build in this order

| # | Component | Notes | Status |
|---|---|---|---|
| 1 | `NbDisplay` | Directive on h1/h2. Mega title — appears in all 18 designs. | Done (code + docs) |
| 2 | `NbSeparator` | Directive on `<hr>`. Horizontal/vertical, solid/dashed/thick. | Done (code + docs) |
| 3 | `NbStatusDot` | Directive on `<span>`. online / offline / live states. | Done (code + docs) |
| 4 | `NbChip` + `NbChipGroup` | Directive + wrapper. Icon slot via `ng-content`. | Done (code + docs) |
| 5 | `NbIconButton` | Directive on `<button>`. Icon via `ng-content`. Shapes: square/circle. | Done (code + docs) |
| 6 | `NbButton` trailing icon slot | Enhancement to existing `NbButton`. Adds `[nbButtonTrailingIcon]` slot. | Done (code) |
| 7 | `NbProgress` | Component. `value`, `max`, `tone`. ARIA progressbar. | Done (code + docs) |
| 8 | `NbAvatarGroup` | Component. Stacked avatars + overflow count (`+142`). | Done (code + docs) |
| 9 | `NbSticker` | Component. Shapes: burst / stamp / pill. Tones + rotation. | Done (code + docs) |
| 10 | `NbHalftone` | Component. Decorative dot grid. Positions: corners. | Done (code + docs) |

### Icons

`NbIconButton`, `NbChip`, and `NbButton` trailing icon all expose `ng-content` slots — bring your own icons (Lucide recommended as interim). Custom brutalist icon set ships as `@ng-brutalism/icons` in v0.2.x.

### Docs-only recipes (no library component)

| Recipe | Built from |
|---|---|
| Profile row | `NbAvatar` + `NbMediaItem` + `NbStatusDot` |
| Feature list | `NbMediaItem` items in a `<ul>` |
| Mini bar chart | Inline SVG snippet |
| Ticket shape / perforated divider | CSS recipe |
| Barcode | CSS recipe |

## Docs pages

### Component docs (14 total)

Write docs pages for all 4 already-done components + 10 new ones. Each page follows the existing pattern: live example, API table, usage code.

### Recipe pages (6 for v0.2.0)

Each recipe page shows: live preview, Angular code, which primitives are used.

| Recipe | Key primitives showcased |
|---|---|
| **Job Card** | Chip, Stat, StatusDot, Separator, IconButton, Sticker, Halftone |
| **Charity Card** | Progress, AvatarGroup, Stat, Chip, Sticker, Halftone |
| **Listing Card** | MediaFrame, Stat, Chip, StatusDot, IconButton, Sticker, Halftone |
| **Esports Card** | Stat, Chip, StatusDot, Tile, Separator, Sticker, Halftone |
| **Profile Card** | Display, Chip, StatusDot, IconButton (social), Sticker |
| **Podcast Card** | Progress, Chip, StatusDot, IconButton, Sticker, Halftone |

These 6 cover all 10 new primitives at least once. Remaining 12 designs ship as recipe pages in follow-up doc drops (v0.2.x).
